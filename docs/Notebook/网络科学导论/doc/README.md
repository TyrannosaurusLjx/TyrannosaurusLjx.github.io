### 传染病模型

种群 ($\text{polulation}$) 有 $N$ 个个体，每个个体有三种状态：易感染 ($\text{Susceptible}$)、感染 ($\text{Infectious}$)、康复 ($\text{Recovered}$)。$S(t)$、$I(t)$、$R(t)$ 分别表示时刻 $t$ 时易感染、感染、康复的个体数。
