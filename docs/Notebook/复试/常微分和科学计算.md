# 常微分 & 科学计算

#### 奇解
如果一个解的每一点上都至少还有另外一个解与之相切, 那么这个解称为奇解

#### 拉格朗日插值

$\phi(x)=\sum_{i=0}^{n}y_iL_i(x)$

其中$L_i(x)=\prod_{j=0,j\neq i}^{n}\frac{x-x_j}{x_i-x_j}$

#### 牛顿插值

$\phi(x)=f(x_0) + f[x_0,x_1](x-x_0)+f[x_0,x_1,x_2](x-x_0)(x-x_1)+\cdots$

### Hermite插值
- 一阶导数也相同

- 还可以分段插值

#### 最小二乘法

$\min \sum_{i=1}^{n}(f(x_i)-y_i)^2$
